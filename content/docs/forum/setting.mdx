# 如何自訂論壇通知設定

論壇允許你決定要透過網頁或電子郵件接收哪些類型的通知。要修改設定,請按以下步驟操作:

1. 登入並點擊頭像,選擇"個人設定"。

2. 向下滾動至"通知"部分。在這裡你會看到各種通知選項。

3. 每個選項都有兩個勾選框 - 一個用於網頁通知,另一個用於電子郵件通知。選中你希望接收的通知類型和傳遞方式。

以下是每個通知類型的說明:

- 你的貼文標題被更改 - 當你發佈的主題標題被版主修改時收到通知
- 當我的回覆被選為最佳回覆時 - 當你的回覆被原貼文作者選為解決方案時收到通知 
- 當我參與的貼文有最佳回覆時 - 當你參與回覆過的主題獲得解決方案時收到通知
- 提醒在我發起的討論中選擇最佳回覆 - 定期收到提醒選擇你發佈主題的最佳回覆
- 有人回覆了你 - 當有人回覆你的評論時收到通知
- 有人提及了你 - 當有人在評論中@提及你的用戶名時收到通知
- 有人提及我所在的群組 - 當有人在評論中提及你所屬群組時收到通知 
- 你的貼文被鎖定 - 當你的主題被版主鎖定並禁止進一步回覆時收到通知

設定完畢後,請記得點擊底部的"保存設定"按鈕。你隨時都可以回到這個頁面修改個人通知偏好。
