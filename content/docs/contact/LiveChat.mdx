---
title: 如何使用 live chat 聯繫我們
description: 如何使用 live chat 聯繫我們
---
如果您在使用我們的產品或服務時遇到任何問題或困難，您可以使用我們的 live chat 功能，即時與我們的客服人員溝通，獲得快速有效的解答和協助。

live chat 是一種網頁即時聊天的功能，讓您可以在我們的官網上，透過文字訊息與我們的客服人員對話，不需要撥打電話或發送電子郵件。您只需要按照以下的步驟，就可以開始使用 live chat 功能：

## 步驟一：開啟 live chat 的視窗
在我們的官網右下角，您會看到一個 live chat 的按鈕，顯示「與我們聊天」或「在線客服」等字樣。請點擊這個按鈕，開啟 live chat 的視窗。

## 步驟二：輸入個人資料
在 live chat 的視窗中，您會被要求輸入一些基本的個人資料，例如您的姓名、電子郵件地址、手機號碼等，以便我們能夠識別您的身份，並在需要時與您聯繫。請填寫這些資料，並按下「開始聊天」的按鈕。

## 步驟三：與機器人或客服人員對話
接下來，您會被連接到一位由 GPT-３ 支援的機器人，並看到它的名字和頭像。您可以在聊天框中輸入您的問題或需求，並按下「發送」的按鈕。我們的機器人會盡快回覆您，並為您提供相關的資訊或建議。如果您想要與真人的客服人員聯繫，請在聊天框中說「我要客服幫忙」，我們的機器人會為您轉接。

## 步驟四：結束聊天並提供回饋
在聊天過程中，您可以隨時點擊聊天框右上角的「X」按鈕，關閉 live chat 的視窗，或者點擊「結束聊天」的按鈕，主動結束聊天。當您結束聊天時，您可以選擇是否要對我們的客服人員或機器人進行評價或留下意見，以幫助我們改善我們的服務品質。

我們的 live chat 功能是一個方便、快捷、免費的聯繫方式，讓您可以隨時隨地與我們的客服人員或機器人交流，解決您在使用我們的產品或服務時遇到的問題或困難。我們非常期待與您聊天，並為您提供最佳的服務體驗。請隨時使用我們的 live chat 功能，與我們聯繫吧！
